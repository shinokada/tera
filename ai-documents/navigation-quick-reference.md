## Tips

### Multiple Ways to Navigate
You can use:
- **Arrow keys** (â†‘â†“) - Standard navigation
- **Vim keys** (jk) - Faster for vim users
- **Numbers** (1-9) - Instant selection
- **Home/End** (gG) - Jump to edges

### Visual Feedback
- Selected items show `>` indicator
- Highlighted in color
- Help text always visible

### Quick Actions
- Press number to instantly select
- Press Enter on highlighted item
- Use Esc to go back
- Use 0 for main menu

### Efficiency
- Use vim keys (jk) for speed
- Use numbers for known items
- Use arrows for browsing
- Use / to filter long lists
